# Bahan Latihan Laravel API - Bahasa Malaysia

Folder ini ialah versi Bahasa Malaysia untuk bahan latihan Laravel API.

Struktur folder dikekalkan sama seperti versi asal:

```text
bahasa-malaysia/
  training/
  examples/
  slides/
```

## Nota Penggunaan

- Teks pengajaran, objektif, tugasan, dan arahan telah dilokalkan ke Bahasa Malaysia.
- Nama class, route, command, config key, field database, token header, dan contoh kod Laravel dikekalkan dalam Bahasa Inggeris supaya kod masih boleh disalin terus ke projek.
- Folder `examples/` masih menggunakan struktur path Laravel sebenar.
- Slide deck Bahasa Malaysia boleh dibuka melalui:

```text
bahasa-malaysia/slides/laravel-api-training.html
```

## Kandungan

- `training/overview.md` - gambaran keseluruhan latihan.
- `training/day-*.md` - tutorial 5 hari.
- `training/react-client-api-setup.md` - setup React/Vite client untuk memanggil Laravel API.
- `training/bonus-*.md` - bonus TDD, Swagger/OpenAPI, FilamentPHP, dan GSD Claude Code prompts.
- `examples/complete-laravel-react-project` - contoh projek lengkap backend Laravel dan frontend React dengan seed data.
- `examples/day-*` - fail contoh untuk setiap hari.
- `examples/react-client-api-consumer` - React/Vite client untuk memanggil REST API.
- `examples/bonus-*` - fail contoh untuk setiap bonus.
- `slides/` - slide reveal.js untuk latihan.
