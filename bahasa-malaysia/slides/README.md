# Slide Latihan Laravel API

Buka slide deck reveal.js utama terus dalam browser:

```text
bahasa-malaysia/slides/laravel-api-training.html
```

## Framework

Deck ini menggunakan reveal.js `6.0.1`, disimpan secara local di:

```text
bahasa-malaysia/slides/vendor/reveal/
```

Ini membolehkan deck dibuka melalui `file://` tanpa CDN atau dev server.

## Controls

- Right arrow, down arrow, Space: slide seterusnya
- Left arrow, up arrow: slide sebelumnya
- Esc: overview slide
- S: speaker notes
- Ctrl/Cmd + F: search
- Alt + click: zoom

## Export Ke PDF

Buka deck dengan:

```text
bahasa-malaysia/slides/laravel-api-training.html?print-pdf
```

Kemudian gunakan browser print dialog dan pilih "Save as PDF".
