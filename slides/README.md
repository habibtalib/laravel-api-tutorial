# Laravel API Training Slides

Open the main reveal.js slide deck directly in a browser:

```text
slides/laravel-api-training.html
```

## Framework

The deck uses reveal.js `6.0.1`, vendored locally under:

```text
slides/vendor/reveal/
```

This keeps the deck usable from `file://` without a CDN or dev server.

## Controls

- Right arrow, down arrow, Space: next slide
- Left arrow, up arrow: previous slide
- Esc: slide overview
- S: speaker notes
- Ctrl/Cmd + F: search
- Alt + click: zoom

## Export To PDF

Open the deck with:

```text
slides/laravel-api-training.html?print-pdf
```

Then use the browser print dialog and choose "Save as PDF".
